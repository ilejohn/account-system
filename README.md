# account-system

- Deploy Note:

copy and paste the content of .env.example into a new file called .env and set values based on your set environment configuration

# Install dependencies
```
npm install
```

# Migrate migration files
```
npm run migrate
```

# To seed database
```
npm run seed
```

# To run on local
```
npm run serve
```

# To run testing
```
npm run test
```
